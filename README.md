# Portifolio

Esse portifolio foi pensado para ser um aplicativo de facil manutenção, testei um sistema diferente de organização do css colocando nomes mais claros em suas classes e dividindo elas em pastas e arquivos de acordo com onde estão no codigo, apesar de precisar colocar bastante arquivos de css no index.html ele acabou facilitando muito quando precisei fazer qualquer alteração para me encontrar no codigo, foi considerada a responsividade porem foi construido em um modelo de desktop first
